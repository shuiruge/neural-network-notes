# Evolution of Neural Network

## File Structure

- notes: TeXmacs files of this note.
  - perceptron.tm: From perceptron to neural network.
  - optimization.tm: Gradient based optimization methods.
  - deep.tm: Deep learning.
  - nlp.tm: Natural language processing.
  - main.tm: As the main file of this note, we put all previous materials into this file.
- public:
  - main.pdf: The PDF output of the note.
- experiments: Numerical experiments.
- drafts: Colletion of drafts

## How to Contribute

- Editing using TeXmacs (together with TeXlive).
